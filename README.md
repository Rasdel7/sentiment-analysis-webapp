 # Sentiment Analysis Web App 😊

Real-time sentiment analyzer that detects positive, negative
and neutral emotions in any text.

## Live Demo
[Click here to try the app](https://sentiment-analysis-webapp-mighw4ywjcbfg3n4yjkikb.streamlit.app)

## Features
- Single text sentiment analysis with polarity gauge
- Bulk analysis of multiple texts at once
- Side-by-side topic sentiment comparison
- Word frequency visualization

## Tools Used
- Python, Streamlit, TextBlob, NLTK, Matplotlib, Pandas

## How to Run Locally
pip install streamlit textblob nltk pandas matplotlib
streamlit run app.py
